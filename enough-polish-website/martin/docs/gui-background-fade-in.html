fade-in Background

  • Related Topics:
  • Backgrounds
  • Create Your Own Background

Fades in a background on MIDP 2.0 handsets.

Design

Quicktime:

Flash: 

.itemMain {
	padding: 2;
}
.itemMain:hover {
	layout: expand | left;
	font-style: bold;
	background {
		type: fade-in;
		target-argb-color: #bfff;
	}
}

CSS attributes:

CSS Attribute  DefaultValuesExplanation
target-argb-color white ARGB color The color of the background when the animation has finished.
restart-on-time false true, false Defines if the animation should be restarted when showNotify is called and the last animation is longer than 5 seconds ago.
restart-on-show true true, false Defines if the animation should be restarted every time when showNotify is called.
