Using the Device Database

J2ME Polish includes a device database, which contains all known J2ME devices and which can be extended easily.
This database can be used for selecting the target devices and for optimizing the application without losing the portability of the application: Any capability or feature of a device can be accessed during the preprocessing phase, an often needed example is checking the API support of a device: 

//#if polish.api.mmapi || polish.midp2
	// this device either supports the 
	// Mobile Media API or the MIDP/2.0 standard
//#endif
