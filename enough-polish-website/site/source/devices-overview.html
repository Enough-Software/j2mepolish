Using the Device Database

J2ME Polish includes a device database, which contains all known J2ME devices and which can be extended easily.
Use this database to select your target devices and to optimize your app, for example with preprocessing: 

//#if polish.api.mmapi || polish.midp2
	// this device either supports the 
	// Mobile Media API or the MIDP/2.0 standard
//#endif
