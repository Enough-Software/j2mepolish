Discussion of J2ME Polish

Forums

You can use the berliOS forum for any discussion. You need to join berliOS for posting messages.

A German Forum is available at the J2MeForum.com. Deutschsprachige User können hier über alles sprechen! Danke an Thomas Polaschek für die Bereitstellung des Forums!

Mailinglists

Please subscribe to one of the following mailing lists:

  • users: please join the polish-users<at>lists<dot>berlios<dot>de-list.
    In this list users can discuss their problems. The developers of J2ME Polish will monitor the discussions and help out!
  • announcements: please join the polish-announce<at>lists<dot>berlios<dot>de-list for retrieving news regarding the polish framework.
    This list anounces new versions and has very low traffic.
  • developers: please join the polish-devel<at>lists<dot>berlios<dot>de-list.
    This is for the developers of J2ME Polish.

Feature-Request

You can now request a new feature for J2ME Polish on berliOS. Please have a look at the roadmap before you add a new request.

Wiki

You can also add comments or help on the J2ME Polish Wiki.
