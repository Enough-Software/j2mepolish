The split text-effect draws a text in two colors that are separated vertically. Combine it with a split or round-rect-split background for best effect.

Usage


The above design has been realized using following CSS settings: 

title {
	margin: 2;
	padding: 2;
	padding-top: 12;
	padding-bottom: 12;
	padding-right: 8;
	font-face: proportional;
	font-size: large;
	font-style: bold;
	font-color: #fff;
	layout: center | expand;
	text-effect: split;
	text-split-bottom-color: #ddd;
	background {
		type: round-rect-split;
		top-color: #073b46;
		bottom-color: #042931;
		split-pos: 50%;
		arc-width: 8;
		arc-height: 8;
	}
}


The top part of the text will be colored in the specified font-color.
