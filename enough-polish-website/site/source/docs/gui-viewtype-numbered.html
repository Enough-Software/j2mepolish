numbered view-type

  • Related Topics:
  • Container View Types
  • View Types
  • Create Your Own View Type
  • Commands Menu

The numbered view-type adds the position of each item as it's label. It is typically used for the commands menu.

Design


Since the numbered view-type adds the numbers as labels, you might want to use the label-style attribute to adjust the style of numbers as shown in the example below: 

menu {
	margin-left: 2;
	padding: 2;
	background {
		type: round-rect;
		color: brightBgColor;
		border-width: 2;
		border-color: bgColor;
	}
	font-color: brightFontColor;
	font-style: bold;
	view-type: slide-up;
	view-type: numbered;
}

menuItem {
	margin-top: 2;
	padding: 2;
	padding-left: 5;
	font {
		color: fontColor;
		size: medium;
		style: bold;
	}
	layout: left;
	label-style: menuLabel;
}

menuItem:hover {
	background-color: bgColor;
	font-color: brightFontColor;
	layout: left | horizontal-expand;
	label-style: menuLabelFocused;
	include-label: true;
}

.menuLabel {
	padding: 3;
	font-color: fontColor;
	font-style: bold;
}

.menuLabelFocused {
	padding: 2;
	font-color: brightFontColor;
	font-style: bold;
	border {
		type: circle;
		width: 1;
		color: brightFontColor;
	}
}
