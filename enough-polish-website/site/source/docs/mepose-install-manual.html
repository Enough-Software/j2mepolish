<%index %>

Prerequisites

You need the following components on your system:

  • Eclipse 3.2 http://www.eclipse.org
    Note: Eclipse 3.1.x is NOT supported.
  • J2ME Polish 2.0 or higher http://www.j2mepolish.org
  • Java Wireless Toolkit, http://java.sun.com/products/j2mewtoolkit/index.html (for Windows and Linux),
    or the Mobile Power Player SDK http://sdk.mpowerplayer.com (for Mac OS X)
  • Java 2 Standard Edition SDK 1.4 or higher, http://java.sun.com/j2se/

Overview

You can install Mepose by using the J2ME Polish installer (when you have writing rights to your Eclipse installation) or by using the Eclipse Update Manager.

Installing Mepose without the update manager is pretty straight forward:

  • Get the plugins archive from http://www.j2mepolish.org/eclipse/de.enough.mepose.zip
  • Unpack the archive in your eclipse directory
  • Call eclipse with the clean parameter to reread the plugin cache

Detailed information for these steps are given below.

Installing Mepose Manually

You need Eclipse 3.2 to get this thing to work. Let's start.

Download all needed plugins and features as a ZIP archive.

After you have downloaded the archive, unpack it into your eclipse directory. E.g. if your eclise installation directory is at C:\Program Files\eclipse, unpack the contents of the archive into this directory.

When you start eclipse the next time, you have to call the parameter clean on its invocation. This need to be done only once. One possible way whould be oben up a console, navigate to the eclipse directory and issue the command.

We assume linux users know how to call their eclipse with a command line parameter :)

You are done.
