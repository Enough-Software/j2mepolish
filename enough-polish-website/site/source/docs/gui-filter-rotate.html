The rotate filter rotates UI elements.

Usage

Use this filter like this (we concentrate on the rotation here): 

.mainItem {
	font-style: bold;
	font-size: medium;
	padding: 0px;
	icon-image: url( icon%INDEX%.png );
	icon-image-align: top;
	layout: center;
	text-filter: rotate;
	text-filter-layout: hcenter | vcenter;
	filter-rotation-angle: -20;
	filter-rotation-angle-animation {
		on: defocus;
		range: 0..-20;
		duration: 400ms;
	}
}

.mainItem:hover {
	font-color: white;
	text-effect: shadow;
	filter-rotation-angle-animation {
		on: focus;
		range: -20..0;
		duration: 400ms;
	}
}
