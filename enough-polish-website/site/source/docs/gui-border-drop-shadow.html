drop-shadow Border

  • Related Topics:
  • Borders
  • Create Your Own Border

Surround an item or a screen with a possibly translucent drop shadow using the drop-shadow border:

Design





.itemMain {
	background-color: #0e0;
	border {
		type: drop-shadow;
		inner-color: #a0e0;
		outer-color: #5050;
		width: 3;
		orientation: all;
	}
}

<%cssmappings border drop-shadow%>
