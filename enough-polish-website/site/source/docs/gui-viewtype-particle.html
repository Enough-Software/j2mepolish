particle view-type

  • Related Topics:
  • View Types
  • Create Your Own View Type

The particle view-type scales an item in an explosion like manner. You can use this to indicate that the user is holding down a button and a corresponding pressed style.

Design

.mainCommand {
	padding: 0;
	padding-vertical: 0;
	icon-image: url( iconlarge%INDEX%.png );
	icon-image-align: top;
	layout: center;
}

.mainCommand:hover {
	font-style: bold;
	font-size: small;
	font-color: white;
	text-effect: shadow;
}

.mainCommand:pressed {
	view-type: particle;
}
