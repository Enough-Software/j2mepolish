Developing for Java SE

You can create a project for Java SE just by adding the device Generic/Java to the list of target devices in your build.xml script.

J2ME Polish will then integrate the microemulator, so that you can just double click the generated JAR file for starting the application - you only need to have Java installed for that.

Known Limitations

There are a couple of limitations of this approach at the moment:

  • MIDlet.getAppProperty(): JAD or MANIFEST attributes are not supported.
  • API Limitations: Currently only following APIs are supported: wmapi, mmapi, btapi and pdaapi.
  • Full Emulator: The emulator menu is visible and no device skin is integrated.
