Fading-Alien-Glow Text-Effect

The fading-alien-glow effect provides an animated glow around the text on MIDP 2.0 devices with floating point support (either by supporting the CLDC 1.1 configuration or by using the Floater tool).

Usage

title {
        //#if polish.midp2 and polish.hasFloatingPoint
			text-effect: fading-alien-glow;
			text-fading-alien-glow-duration-in: 15;
	        text-fading-alien-glow-outer-in-color: #ff80ff00;
	        text-fading-alien-glow-outer-out-color: #ff0080ff;
	        text-fading-alien-glow-steps: 4;
        //#endif
        layout: horizontal-center | horizontal-expand;
        font {
                face: proportional;
                size: large;
                style: bold;
                color: #fff;
        }
        background-color: #fff;
}
