Assembling of Resources and Localization

J2ME Polish assembles resources and creates localized versions of your application automatically.

The actual resource assembling is quite easy - just put all general resources into the "resources" folder of your project and put specialized resources into the appropriate subfolder, e.g. all Nokia-resources into "resources/Nokia", any Siemens-SX1-resources into "resources/Siemens/SX1", any MP3 files into "resources/mp3" and any high-color images into "resources/BitsPerColor.16+".
This process can be fine tuned using the <resources> element so that for example MIDI files are only added when the device does not support MP3 and so forth.

J2ME Polish has the best ever localization framework which has absolutely no overhead in optimal cases, since localized messages are actually embedded directly in the source-code.

Going further:

  • Localization describes how applications can be localized easily and effectively using J2ME Polish.
