video Background

  • Related Topics:
  • Backgrounds
  • Create Your Own Background

Displays videos on devices with MMAPI support. Note that nothing can be painted in front of the video, as this is usually rendered natively and directly by the operating system.

Design

title {
	font-color: black;
	//#if polish.api.mmapi
		background {
			type: video;
			color: white;
			video: url( dj.mpeg );
			mime-type: video/mpeg2;
			anchor: right | vertical-center;
			x-offset: 3;
		}
	//#endif
}

CSS attributes:

<%cssmappings background video %>
