The typewriter effect writes one character after the other as if you are typing it on a typewriter.

Usage

title {
        text-effect: typewriter;
        layout: horizontal-center | horizontal-expand;
        font {
                face: proportional;
                size: large;
                style: bold;
                color: #fff;
        }
        background-color: #fff;
}
