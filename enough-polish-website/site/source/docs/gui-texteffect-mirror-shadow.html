The mirror-shadow text-effect draws a mirrored copy under the text. This effect requires MIDP 2.0 support.

Usage


The above effect has been realized for the screen's title with following settings: 

title {
        //#if polish.midp2
                text-effect: mirror-shadow;
        //#endif
        layout: horizontal-center | horizontal-expand;
        font {
                face: proportional;
                size: large;
                style: bold;
                color: #eef1e5;
        }
        background {
                type: simple;
                color: #ffffffff;
        }
}
