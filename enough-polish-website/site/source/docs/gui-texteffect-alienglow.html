The alien-glow effect provides an eerie glow around the text on MIDP 2.0 devices with floating point support (either by supporting the CLDC 1.1 configuration or by using the Floater tool).

Usage


title {
        //#if polish.midp2 and polish.hasFloatingPoint
               text-effect: alien-glow;
        //#endif
        layout: horizontal-center | horizontal-expand;
        font {
                face: proportional;
                size: large;
                style: bold;
                color: #fff;
        }
        background-color: #fff;
}

Specific design attributes of the Alien-Glow Text-Effect:

CSS Attribute  DefaultValuesExplanation
text-alien-glow-inner-color #FFFFFFFF ARGB color definition The inner part color of the glow. 
text-alien-glow-inner-color: #c00f;
text-alien-glow-outer-color #FF00FF00 ARGB color definition The outer part color of the glow. 
text-alien-glow-outer-color: #af00;
