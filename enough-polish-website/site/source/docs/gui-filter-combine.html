Combine RGB Filters

  • Related Topics:
  • Other Filters
  • CSS Animations
  • GUI Concepts
  • CSS Basics

Here we combine the scale and grayscale RGB filters. When an item is focused, the image will be scaled up and colors will be restored. When an item is defocused, it will be scaled down and colors will be removed.

Usage

Just use a comma separated list of filters for combining them, e.g. filter: scale, blur;. 

.mainItem {
	font-style: bold;
	font-size: medium;
	padding: 0px;
	icon-image: url( icon%INDEX%.png );
	icon-image-align: top;
	layout: center;
	
	icon-filter: scale, grayscale;
	filter-scale-grade: 60%;
	filter-scale-grade-animation {
		on: defocus;
		range: 100%..60%;
		duration: 400ms;
	}
	filter-grayscale-grade: 100%;
	filter-grayscale-grade-animation {
		on: defocus;
		range: 0%..100%;
		duration: 400ms;
	}
	
}

.mainItem:hover {
	font-color: white;
	text-effect: shadow;
	
	filter-scale-grade-animation {
		on: focus;
		range: 60%..100%;
		duration: 400ms;
	}
	filter-grayscale-grade-animation {
		on: focus;
		range: 100%..0%;
		duration: 400ms;
	}		
	
}
