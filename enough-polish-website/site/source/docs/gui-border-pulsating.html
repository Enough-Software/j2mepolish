pulsating Border

  • Related Topics:

Draw an animated rectangular border around the UI component using the pulsating border:

Design

.itemMain {
	border {
		type: pulsating;
		start-color: #0f0;
		end-color: #000;
		back-and-forth: true;
		repeat: true;
	}
}
