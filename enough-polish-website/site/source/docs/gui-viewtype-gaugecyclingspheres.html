gauge-cycling-spheres view-type

  • Related Topics:
  • Gauge
  • Gauge View Types

The gauge-cycling-spheres view-type displays several filled circles in a cycle while exchanging the color of one of them for a Gauge.

Design

.gaugeCyclingSpheres 
{
	view-type: gauge-cycling-spheres;
	gauge-cycling-spheres-count: 8;
	gauge-cycling-spheres-highlight-count: 3;
	gauge-cycling-spheres-color: black;
	gauge-cycling-spheres-highlight-color: #96B400;
	gauge-cycling-spheres-highlight-center-color: #B93610;
	gauge-cycling-spheres-interval: 50;
	layout: horizontal-center | horizontal-expand;}
