The rotating effect rotates text.

Usage

title {
     //#if polish.midp2 and polish.hasFloatingPoint
         text-effect: rotating;
     //#endif
     layout: horizontal-center | horizontal-expand;
     font {
         face: proportional;
         size: large;
         style: bold;
         color: #333;
     }
     background-color: #fff;
}
