ChoiceGroup: horizontal-choice view-type

The horizontal-choice view-type shows all options of an exclusive ChoiceGroup in a row.

Design

.horizontalChoice {
	label-style: choiceGroupLabel;
	background-type: round-rect;
	background-color: focusedBgColor;
	include-label: true;
	layout: expand;
	view-type: horizontal-choice;
	horizontalview-left-arrow: url( arrow-left.png );
	horizontalview-right-arrow: url( arrow-right.png );
}

.horizontalChoice:hover {
	background-color: white;
}

.choiceItem {
	font-color: gray;
	font-style: bold;
	text-wrap: false;
}

.choiceItem:hover {
	font-color: black;
}

.choiceGroupLabel {
	font-style: italic;
	font-size: small;
	layout: newline-after;
}
