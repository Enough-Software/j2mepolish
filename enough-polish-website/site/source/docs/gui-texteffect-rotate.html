The rotate text-effect rotates the text. This effect requires MIDP 2.0 with floating point support (either by supporting the CLDC 1.1 configuration or by using the Floater tool).

Usage


The above effect has been realized for the screen's title with following settings: 

title {
        //#if polish.midp2 and polish.hasFloatingPoint
			text-effect: rotate;
			text-rotate-angle: 8;
        //#endif
        layout: horizontal-center | horizontal-expand;
        font {
			face: proportional;
			size: large;
			style: bold;
			color: #333;
        }
        background {
			type: simple;
			color: #ffffffff;
        }
}
