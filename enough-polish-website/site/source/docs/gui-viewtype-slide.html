slide view-type

  • Related Topics:
  • View Types
  • Create Your Own View Type
  • Commands Menu

The slide view-type moves the affected item in the specified direction, by default upwards. It is typically used for the commands menu.

Design

menu {
	margin-left: 2;
	padding: 2;
	background {
		type: round-rect;
		color: brightBgColor;
		border-width: 2;
		border-color: bgColor;
	}
	font-color: brightFontColor;
	font-style: bold;
	menubar-color: bgColor;
	view-type: slide;
}
