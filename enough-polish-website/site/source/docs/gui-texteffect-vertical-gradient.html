The vertical-gradient text-effect draws text with a vertical-gradient.

Usage


The above effect has been realized for the screen's title with following settings: 

title {
        text-effect: vertical-gradient;
        text-vertical-gradient-end-color: #ff0099;
        text-vertical-gradient-start-color: #ffffff;
        layout: horizontal-center | horizontal-expand;
        font {
                face: proportional;
                size: large;
                style: bold;
        }
        background-color: white;
}
