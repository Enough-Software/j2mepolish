Horizontal-Gradient Background

  • Related Topics:
  • Backgrounds

Design

The vertical-gradient background paints a linear, horizontal gradient. 

.myItem {
	type: radial-gradient;
	outer-color: blue;
	inner-color: white;
}

You can also adjust the center position of the gradient using the center-x and center-y attributes, which are given in percentage - 0 corresponds to the center, -100 corresponds to the most left or top edge of the square that surrounds the circle, +100 the right/top edge: 

.myItem {
	type: radial-gradient;
	outer-color: blue;
	inner-color: white;
	center-x: -70;
	center-y: -70;
}

.myItem {
	type: radial-gradient;
	outer-color: blue;
	inner-color: white;
	center-x: -100;
	center-y: -100;
}

.myItem {
	type: radial-gradient;
	outer-color: blue;
	inner-color: white;
	center-x: 0;
	center-y: -130;
}
