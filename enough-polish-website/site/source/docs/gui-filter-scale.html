scale filter

The scale filter scales UI elements up or down.

Usage

Use this filter like this: 

.mainItem {
	font-style: bold;
	font-size: medium;
	padding: 0px;
	icon-image: url( icon%INDEX%.png );
	icon-image-align: top;
	layout: center;
	icon-filter: scale;
	filter-scale-grade: 60%;
	filter-scale-grade-animation {
		on: defocus;
		range: 100%..60%;
		duration: 400ms;
	}
}

.mainItem:hover {
	font-color: white;
	text-effect: shadow;
	filter-scale-grade-animation {
		on: focus;
		range: 60%..100%;
		duration: 400ms;
	}
}
