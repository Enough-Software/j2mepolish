gif89a Background

  • Related Topics:
  • Backgrounds
  • Create Your Own Background

Display animated GIF89a images on devices with MMAPI support. Note that nothing can be painted in front of the animated image, as this is usually rendered natively and directly by the operating system.

Design

title {
	font-color: black;
	//#if polish.api.mmapi
		min-height: imageheight( love.gif );
		background {
			type: animated-gif;
			color: white;
			image: url( love.gif );
			anchor: right | vertical-center;
			x-offset: 3;
		}
	//#endif
}

CSS attributes:

<%cssmappings background gif89a %>
