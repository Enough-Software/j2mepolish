horizontal-split Background

  • Related Topics:
  • Backgrounds
  • Create Your Own Background

The horizontal-split background separates the background in a left and a right part.

Design


The above title design has been realized by combining a horizontal-split with a circle background: 

backgrounds {
	titleCircle {
		type: circle;
		color: red;
		diameter: 80%;
		anchor: right | vcenter;
		x-offset: -5;
	}
	titleBg {
		type: horizontal-split;
		left-color: #f00;
		right-color: #ccc;
		split-pos: 75%;
	}
}
title {
	margin: 2;
	padding: 2;
	padding-top: 12;
	padding-bottom: 12;
	padding-left: 12;
	padding-right: 8;
	font-face: proportional;
	font-size: large;
	font-style: bold;
	font-color: #ccc;
	background {
		type: combined;
		foreground: titleCircle;
		background: titleBg;
	}
	layout: left | expand;	
}

CSS attributes:

<%cssmappings background horizontal-split %>
