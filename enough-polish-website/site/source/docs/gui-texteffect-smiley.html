The smiley text-effect replaces textual smileys like :-) with smiley images.

Usage


The above effect has been realized for the input field with following settings: 

.itemInput {
        text-effect: smiley;
        layout: horizontal-center | horizontal-expand;
        font {
           face: proportional;
           size: large;
           style: bold;
           color: black;
        }
        background-color: #7196b5;
}

Additionally following settings have been made in the build.xml script: 

<variable file="configuration/smileys.properties" />

In the corresponding configuration/smileys.properties following settings are made: 

smileys={new Smiley(new String[]{":-)",":)"},"/smiley_1.png","smile"),new Smiley(new String[]{":-P",":P"},"/smiley_2.png","cheekie"),new Smiley(new String[]{":-(",":("},"/smiley_3.png","ouch")}

Note that the definitions are made within a properties file rather than directly within the build.xml script, since we use many quotation-marks which otherwise would need to be quoted as &quot; in the build.xml script.

For each smiley you need to add a smiley constructor with following parameters:

  1. String[] smileys: an array of textual representations that should be replaced
  2. String imageUrl: the URL of the corresponding smiley image
  3. String description: optional name of the smiley

IMPORTANT: usage might/will probably change with the J2ME Polish 2.1.
