Porting

Porting is done in J2ME Polish in various ways:

  • Rich APIs: our APIs are programmed in a defensive way and adapt themselves automatically to different devices, platforms and known issues.
  • PostCompiler: PostCompiler use compiled classes and search for problematic code. An specific example includes accessing System.arrayCopy() on a Nokia N80 with Vodafone firmware version - such devices behave very badly to such calls and thus we replace them during the build process.
  • Preprocessing: You can adjust and control your source code using Preprocessing.
