round-tab Background

  • Related Topics:
  • Backgrounds
  • Create Your Own Background
  • TabbedForm

Displays a background with rounded corners at the top. This can be especially useful for designing the tabs of a TabbedForm.

Design

.itemMain {
	background {
		type: round-tab;
		color: #0e0;
		arc-width: 6;
		arc-height: 6;
	}
}

CSS attributes:

<%cssmappings background round-tab %>
