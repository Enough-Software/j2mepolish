The segment text-effect segments the text.

Usage


The above effect has been realized for the screen's title with following settings: 

title {
        text-effect: segment;
        text-segment-color: white;
        text-segment-gap: 3;
        layout: horizontal-center | horizontal-expand;
        font {
           face: proportional;
           size: large;
           style: bold;
           color: #333;
        }
        background-color: white;
}
