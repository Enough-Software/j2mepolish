Snowflakes Background

  • Related Topics:
  • Backgrounds

Design

The snowflakes background shows animated slowflakes with a background color: 

.myItem {
	background {
		type: snowflakes;
		color: red;
		flake-color: white;
		max-flake-size: 12;
		number-of-flakes: 15;
	}
}

CSS attributes:

CSS Attribute  DefaultValuesExplanation
colorredcolor definition or transparent. The color of the background, can be a rbg or hexadecimal value of the color, the name of the color as defined in the colors section or just transparent in case you are using the background in a combined background.
flake-colorwhitecolor The color of the snowflakes
max-flake-size10integer The maximum size of snowflakes
number-of-flakes5integer The number of flakes that are being animated
