NetBeans Integration

  • Related Topics:
  • Installation of J2ME Polish
  • Installation of the NetBeans Module
  • Creating a new Project
  • Running and Debugging

J2ME Polish provides a NetBeans module that allows you to create J2ME projects that use J2ME Polish for building and that also transfer all device database information from your target devices into NetBeans, so that you can use auto-completion within preprocessing, for example.

After the installation you can create a new J2ME Polish project by selecting File > Project > Mobile > J2ME Polish.
