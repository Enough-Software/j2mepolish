gauge-cycling-icons view-type

  • Related Topics:
  • Gauge
  • Gauge View Types

The gauge-cycling-icons view-type displays several images in a cycle while exchanging one of them for a Gauge.

Design

.gaugeCyclingIcons 
{
	view-type: gauge-cycling-icons;
	gauge-cycling-icons-count: 8;
	gauge-cycling-icons-image: url( sphere.png );
	gauge-cycling-icons-highlight-image: url( sphere_highlight.png );
	gauge-cycling-icons-highlight-count: 1;
	gauge-cycling-icons-interval: 50;
	layout: horizontal-center | horizontal-expand;
}
