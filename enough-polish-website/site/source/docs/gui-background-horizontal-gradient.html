Design

The vertical-gradient background paints a linear, horizontal gradient. 

.myItem {
	type: horizontal-gradient;
	left-color: red;
	right-color: white;
	start: 0%;
	end: 100%;
}

CSS attributes:

<%cssmappings background horizontal-gradient %>
