gauge-horizontal-spheres view-type

  • Related Topics:
  • Gauge
  • Gauge View Types

The gauge-horizontal-spheres view-type displays several filled circles in a row while exchanging the color of one of them for a Gauge.

Design

.gaugeHorizontalSpheres 
{
	view-type: gauge-horizontal-spheres;
	gauge-horizontal-spheres-count: 16;
	gauge-horizontal-spheres-highlight-count: 1;
	gauge-horizontal-spheres-color: gray;
	gauge-horizontal-spheres-highlight-color: blue;
	gauge-horizontal-spheres-interval: 700;
	layout: horizontal-center | horizontal-expand;
}
