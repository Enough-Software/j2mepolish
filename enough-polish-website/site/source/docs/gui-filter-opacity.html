opacity filter

  • Related Topics:
  • Other Filters
  • CSS Animations
  • GUI Concepts
  • CSS Basics

The opacity filter adds translucency for UI elements.

Usage

Use this filter like this: 

.mainItem {
	font-style: bold;
	font-size: medium;
	padding: 0px;
	icon-image: url( icon%INDEX%.png );
	icon-image-align: top;
	layout: center;
	icon-filter: opacity;
	filter-opacity: 20%;
	filter-opacity-animation {
		on: defocus;
		range: 100%..20%;
	}
}

.mainItem:hover {
	font-color: white;
	text-effect: shadow;
	filter-opacity-animation {
		on: focus;
		range: 20%..100%;
	}
}
