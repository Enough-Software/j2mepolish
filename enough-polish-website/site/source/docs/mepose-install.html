Installing Mepose

<%index %>

Prerequisites

You need the following components on your system:

  • Eclipse 3.2 or higher http://www.eclipse.org
  • J2ME Polish 2.0 or higher http://www.j2mepolish.org
  • A device emulator
    for Windows and Linux Sun Java Wireless Toolkit
    http://java.sun.com/products/sjwtoolkit/index.html
    for Mac OSX Mobile Power Player SDK
    http://mpowerplayer.com

Using the J2ME Polish Installer

During the installation of J2ME Polish you can specify the installation directory of Eclipse by setting the eclipse.home property in the Paths dialog:
On the next page, select 'Eclipse integration' by ticking the corresponding checkbox:

Using the Eclipse Update Manager

Open the update manager from the menu with "Help/Install New Software...".
Enter the URL http://www.j2mepolish.org/mepose as an update site.
Choose the Mepose Feature. Click "Next" and follow the steps of the wizard to finish the installation.

Setting up Eclipse

Open Window Preferences and go to the "Debug" panel. Uncheck the item "Suspend execution on uncaugt exceptions"

The installation is done!
