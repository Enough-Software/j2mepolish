Round-Rect Background

  • Related Topics:
  • Backgrounds

Design

The round-rect background displays a monocolored background with rounded corners. 

.myItem {
	background {
		type: round-rect;
		arc: 9;
		color: red;
	}
}

CSS attributes:

CSS Attribute  DefaultValuesExplanation
typen/around-rect The type setting is required for selecting a round-rect background.
colorwhitecolor definition The color of the background, can be a rbg or hexadecimal value of the color or the name of the color as defined in the colors section.
arc10numerical value The diameter of the arc at the four corners.
arc-width10numerical values The horizontal diameter of the arc at the four corners. Defaults to the arc-value, when none is specified.
arc-height10numerical values The vertical diameter of the arc at the four corners. Defaults to the arc-value, when none is specified.
