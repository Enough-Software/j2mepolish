round-rect-split Background

  • Related Topics:
  • split text-effect
  • Backgrounds
  • Create Your Own Background

Paint a background with rounded corners and two colors using the round-rect-split background.
For improving the effect you can bundle this with the split text-effect.

Design


The above design has been realized using following CSS settings: 

title {
	margin: 2;
	padding: 2;
	padding-top: 12;
	padding-bottom: 12;
	padding-right: 8;
	font-face: proportional;
	font-size: large;
	font-style: bold;
	font-color: #fff;
	layout: center | expand;
	text-effect: split;
	text-split-bottom-color: #ddd;
	background {
		type: round-rect-split;
		top-color: #073b46;
		bottom-color: #042931;
		split-pos: 50%;
		arc-width: 8;
		arc-height: 8;
	}
}

CSS attributes:

<%cssmappings background round-rect-split %>
