round-tab-bordered Background

  • Related Topics:
  • Backgrounds
  • Create Your Own Background
  • TabbedForm

Displays a background with rounded corners at the top. This can be especially useful for designing the tab-bordereds of a TabbedForm.

Design

.itemMain {
	background {
		type: round-tab-bordered-bordered;
		color: #0e0;
		arc-width: 6;
		arc-height: 6;
		border-color: #8f8;
		border-width: 1;
	}
}

CSS attributes:

<%cssmappings background round-tab-bordered %>
