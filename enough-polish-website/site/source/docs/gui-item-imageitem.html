The ImageItem displays an image, or if that is not defined, an alternative text.

Design

You can use following attributes for designing a ImageItem:

<%cssattributes de.enough.polish.ui.ImageItem %>

Programming

Use the ImageItem like the MIDP javax.microedition.lcdui.ImageItem: 

//#style imageItem
this.imageItem = new ImageItem( null, image, Item.LAYOUT_LEFT, "Splash!", Item.BUTTON );
this.form.append( this.imageItem );

When an image is to wide for the screen or the chosen layout, it will automatically be interactive. The user can now scroll the image horizontally by pressing left/right on the keyboard or by tapping against the edges of the image.
