Display an image together with text using the IconCustomItem.

Design


The above design has been realized with these CSS settings: 

.myIconItem {
	font-color: rgb( 30, 85, 86 );
	icon-image-align: bottom;
}

IconCustomItem CSS Attributes

Use following attributes to design IconCustomItems:

<%cssattributes de.enough.polish.ui.IconItem %>

Programming

Programming the IconCustomItem is trivial as you can only specify the label, text and icon-image: 

Image startGameImage = Image.createImage("/icon_start.png");
//#style myIconItem
IconCustomItem item = new IconCustomItem(null, "Start Game", startGameImage );
this.form.append( item );
