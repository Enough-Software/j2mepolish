gauge-horizontal-icons view-type

  • Related Topics:

  • Gauge View Types

  • Create Your Own View Type

The gauge-horizontal-icons view-type displays several images in sequence while viewing one image at a time.

Design

.gaugeHorizontalIcons 
{
	view-type: gauge-horizontal-icons;
	gauge-horizontal-icons-count: 8;
	gauge-horizontal-icons-image: url( arrow.png );
	gauge-horizontal-icons-highlight-image: url( arrow_highlight.png );
	gauge-horizontal-icons-highlight-count: 1;
	gauge-horizontal-icons-interval: 500;
	layout: horizontal-center | horizontal-expand;
}
