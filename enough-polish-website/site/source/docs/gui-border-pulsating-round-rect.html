pulsating-round-rect Border

  • Related Topics:
  • Borders
  • Create Your Own Border

Draw an animated border with rounded corners around the UI component using the pulsating-round-rect border:

Design

.itemMain {
	border {
		type: pulsating-round-rect;
		start-color: #0f0;
		end-color: #000;
		back-and-forth: true;
		repeat: true;
	}
}

CSS attributes:

<%cssmappings border pulsating-round-rect%>
