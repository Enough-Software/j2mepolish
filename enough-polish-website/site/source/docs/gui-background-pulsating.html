Display an background that changes its color over time.

Design

}

.itemMain:hover {
	layout: expand | left;
	font-style: bold;
	background {
		type: pulsating;
		start-color: #fff;
		end-color: #0e0;
		steps: 6;
		back-and-forth: true;
	}
}
