Platforms

J2ME Polish can be adjusted to support several platforms that support more than the normal MIDP profile. The key for this support is the ${polish.home}/platforms.xml file (visible only if you install the external device database), inclusion of platform specific (boot) classpaths, and the automatic selection and invocation of preprocessors, postcompilers, finalizers and so on.

Following Platforms are supported by J2ME Polish:

  • Android
  • BlackBerry
  • J2ME/Java ME MIDP
  • Java SE
  • Windows Phone (with additional licensing/services)
  • iOS (iPhone and iPad) (with additional licensing/services)
