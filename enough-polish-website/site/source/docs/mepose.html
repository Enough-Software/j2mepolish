Mepose

Mepose is the Eclipse environment for using J2ME Polish. It helps you setup a new J2ME Polish project and run it with different device settings.

Current Status

Please note that Mepose is currently under development and should not be used within production environments right now. If you have any questions or comments, please email us or submit your questions to the discussion board.
