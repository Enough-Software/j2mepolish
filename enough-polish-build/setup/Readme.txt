*** J2ME Polish
****************************************************

Find more information about J2ME Polish at http://www.j2mepolish.org

The documentation is located in the "doc" folder.

Licenses of J2ME Polish as well as included third party software are located in the "legal" folder.

This software is OSI Certified Open Source Software.
OSI Certified is a certification mark of the Open Source Initiative.
(http://http://www.opensource.org).

Commercial licenses for closed source software are also available,
please check http://www.j2mepolish.org.


****************************************************
*** Sample Applications
****************************************************

You can find sample applications in the "${polish.home}/samples folder.
When you use the command line just call "ant" for building the applications in the appropriate folder
or call "ant emulator" for building the application and launching the emulator.

Have fun!
