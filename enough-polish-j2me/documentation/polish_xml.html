polish.xml

The polish.xml file describes the project and is used to define all relevant properties and variables.

Following tags are used:

TagAttributesdescription
imageLoadStrategy background Determines whether images of Widgets, Backgrounds and Canvasses should be loaded in the background or in the foreground thread. The background-attribute should be either set to "yes" or "no".
When images are loaded in the background the responsiveness of an application can improve dramatically, but it also means that some images are set while the user is in a screen.
When images are loaded in the foreground, all images will be shown when the user is entering a screen. The responsiveness of the application might suffer, but the screens will not change without the user's input.
Example:
<imageLoadStrategy background="yes" />
